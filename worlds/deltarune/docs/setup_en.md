# What you're gonna need:

DELTARUNE Version 1.04. If you're on the beta branch or if your game is modded, it won't work properly. ( https://store.steampowered.com/app/1671210/DELTARUNE/ )
This APWorld does not work with the Chapter 1 & 2 Demo.

The latest DELTARUNE APWorld. The patch files are included in this APWorld, so you won't have to download any mod loader for now.
**This does mean you will have to repatch every time you install a new version of the APWorld.**

It's recommended to use Universal Tracker to help with finding items, but it isn't necessary.
There is no Poptracker for this APWorld yet.
Universal Tracker download (There will be a new tab if you have it in the custom worlds folder): https://github.com/FarisTheAncient/Archipelago/releases

# Setup guide:

**LINUX: Archipelago doesn't like the way a lot of .exe runners try to connect to the servers. We're working on some possible fixes, but you will probably encounter issues such as frequent disconnects that make the game pretty much unplayable. Stay tuned for when we get linux support working! But also sorry if you were looking forward to playing this rn :(**
 
1. Download the APWorld and place it into archipelago/custom_worlds (Double clicking the APWorld will also work!)

2. Run Archipelago Launcher (Or restart the launcher if it's already running)

3. Click "Generate Template Options", find DELTARUNE.yaml, and save that for your settings. Alternatively, use the Options Generator tab in the launcher. Be warned that using "Options Generator" will make it harder to read some of the option descriptions. Otherwise it works great.

4. Run the DELTARUNE client and in the client type `/auto_patch steaminstall` . There should now be a new DELTARUNE folder in archipelago folder. **If it throws an error, try:** In steam, go to manage, and browse local files, copy the folder path, Type `/auto_patch "[insert folder path]"`

5. To connect to a Multiworld, open the game and connect using your port, username, and (if applicable) password. You can also connect with the DELTARUNE client for text chat and Universal Tracker support (if you have the tracker APWorld installed), but connecting with the client isn't required.

7. Make sure to read the README too! It has a lot of information on the stuff the mod changes.
